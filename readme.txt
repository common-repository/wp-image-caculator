== Description ==

The easiest way to calculate the images you have in wordpress media directory.

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Media > WP Image Calc.
4. Press the 'Calculate Images' button.

== Frequently Asked Questions ==

**From where the WP Image Calculator search for images?**

It will search at WordPress defined media directory i.e. wp-content/uploads
